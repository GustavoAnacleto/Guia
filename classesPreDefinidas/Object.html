Object

A classe 'Object' é a classe base de todas as outras classes em Java, direta ou indiretamente estendem(herdam) da classe 'Object'. Portanto, 'Object' é a superclasse de todas as classes em Java.

A classe 'Object' fornece um conjunto de métodos que estão disponíveis para todas as classes em Java. Aqui abaixo estão alguns dos métodos mais comuns fornecidos pela classe 'Object':

  • toString(): Retorna uma representação de string do objeto.

    • O método toString() é um método em Java que pertence à classe Object. Ele é usado para retornar uma representação de string do objeto. Quando você imprime um objeto usando System.out.println() ou concatena-lo com uma string usando o operador +, o método toString() é automaticamente chamado para obter uma representação legível do objeto em forma de string.

  • equals(Object obj): verifica se o objeto atual é igual a outro objeto especificado.

    • O método equals(Object obj) é um método em Java que é usado para verificar se dois objetos são iguais. Ele compara o objeto atual com o objeto passado como argumento (obj) para determinar se eles representam a mesma entidade.
    Por padrão, a implementação do método equals() na classe Object verifica se os dois objetos referem-se ao mesmo local de memória, ou seja, se são exatamente o mesmo objeto.   • hashCode():

    • O método hashCode() é um método em Java que retorna um valor numérico que representa o código de dispersão (hash code) de um objeto. Este código de dispersão é usado principalmente em estruturas de dados como tabelas de hash para melhorar a eficiência de operações de busca, como aquelas implementadas em coleções como HashMap, HashSet, etc. A implementação padrão do método hashCode() na classe Object retorna um código de dispersão baseado no endereço de memória do objeto, o que significa que dois objetos diferentes terão quase certeza de ter códigos de dispersão diferentes, mesmo que sejam iguais de acordo com o método equals(). Ao sobrescrever o método hashCode() em uma classe personalizada, é importante garantir que ele seja consistente com o método equals(). Isso significa que dois objetos que são iguais de acordo com o método equals() devem produzir o mesmo código de dispersão.
