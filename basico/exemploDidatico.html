Workspace

Importante selecionar a pasta onde os projetos serão salvos e tomar cuidado para não se perder quando precisar abrir o projeto A ou B, normalmente se não se atentar a isso fica tudo desorganizado, vamos seguir em frente.

Dicas sobre a ferramenta

•Workspace(selecione a pasta aonde os projetos serão salvos)
•Mudar o layout: Window-> Perspective -> Open Perspective -> Java
•Zerar o layout: Window-> Perspective -> Reset Perspective
•Mostrar a aba Console: Window-> Show View-> Console
•Criar projeto: File -> New -> Java Project
