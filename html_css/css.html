Introdução ao CSS

Mostrando a seguir o código CSS na íntegra dividido em Três partes, logo após o detalhamento de cada tag.

Explicando tag a tag

=> *{ margin: 0; border: 0; padding: 0; }: Esta regra define que todos os elementos HTML devem ter margem, borda e preenchimento igual a zero. Isso remove espaços extras que podem aparecer por padrão em alguns navegadores.

=> html{ ... }: Define estilos para o elemento html>, como a fonte padrão e o tamanho de fonte base. Isso afeta todo o conteúdo na página, a menos que seja substituído por estilos mais específicos.

=> header{ ... }: Estilos específicos para o cabeçalho da página, como cor de fundo, cor do texto, tamanho de fonte e preenchimento. A posição fixa mantém o cabeçalho no topo da página enquanto o usuário rola para baixo.

=> footer{ ... }: Estilos semelhantes ao cabeçalho, mas para o rodapé da página.

=> body{ ... }: Define o plano de fundo do corpo da página.

=> figure{ ... }: Estilos para a tag figure>, que geralmente envolve imagens e suas legendas. Neste caso, o texto é centralizado.

=> #caixa{ ... }: Define estilos para um elemento com o ID caixa, como cor de fundo, tamanho e posição. Este elemento parece ser uma caixa flutuante no canto superior direito da página.

=> Estilos para títulos de diferentes níveis (h1, h2, h3, h4, h5) definem o tamanho da fonte para cada nível de cabeçalho.

=> .conteudo e .conteudo-adaptado: Estilos para seções de conteúdo, como cor de fundo, tamanho de fonte, preenchimento e margem.

=> .imagens-site: Estilos para imagens, como borda arredondada e tamanho.

=> @media(min-width: 280px) and (max-width: 560px){ ... }: Define estilos específicos para telas de tamanho pequeno a médio, usando consultas de mídia. Isso permite que a página seja responsiva e se ajuste a diferentes tamanhos de tela.

=> #grid-conteudo e .grid div: Estilos para criar uma grade de conteúdo, usando a propriedade display: grid.

=> #formulario e estilos para elementos de formulário como inputs, botões e textarea.

=> nav e a: Estilos para a navegação, como alinhamento de texto, espaçamento entre palavras e cor do texto.

=> a:hover: Define o estilo de um link quando o mouse está sobre ele, mudando a cor do texto para amarelo. Isso fornece feedback visual ao usuário quando eles passam o mouse sobre os.
